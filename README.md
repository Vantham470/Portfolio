# Thieng Vanntham — Developer Portfolio (Static)

Pure HTML, CSS, and JavaScript — no Django, no build step.

## Run

Just open `index.html` in a browser.

Or serve locally (recommended so fonts/icons load cleanly):

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000
```

## Customize

- Replace `images/avatar.jpg` with your photo (or leave blank — initials show).
- Put your CV at `files/resume.pdf` (create the `files/` folder).
- Edit text, projects, and skills directly in `index.html`.
- Change colors in the `:root` block at the top of `css/style.css`.

## Structure
